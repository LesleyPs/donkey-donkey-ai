# donkey-donkey
Donkey Donkey AI Travel 🫏✈️The "Beast of Burden" for the Modern Traveler.Donkey Donkey is a specialized AI travel engine designed to act as a senior guide. It moves beyond generic search by performing deep vetting—checking for safety, noise levels, and specific traveler needs—and presenting results with a human-centric "verdict."🎯 The Core MissionTo solve the "Split-Brain UI" problem. Instead of a disconnected chat window next to a search list, Donkey Donkey uses Incremental Discovery: showing immediate, high-quality results for vague prompts and using lightweight, integrated dialogue to refine the trek.🎨 Brand DNAThe Persona: A senior-level guide—handsome, helpful, and protective.Typography: Golos Text. Chosen for high-performance legibility and modern personality.Design Language: "Cartoon Premium" aesthetic with 8px border radius tokens and bouncy spring physics.Color Palette: Midnight Burro (#331800), Sky Trot Blue (#7AB9D2), and Action Blue (#2563eb).🗺️ Key UX FeaturesThe Verdict: Every card leads with a vetted summary explaining why it fits your specific vibe.Unified Feed: Conversation happens naturally below results for a seamless mobile experience.Neighborhood Intelligence: District-level breakdowns with clear trade-offs (e.g., +Walkable, -Noisy).Beast of Burden Input: A prominent natural language input that replaces traditional dropdowns and filters.🌍 Cultural Guardrail: Two-Spirit IdentityDonkey Donkey maintains an absolute distinction between Two-Spirit (an Indigenous spiritual and traditional role) and Western gender identities. We respect cultural specificity in all vetting and results.🚀 Deployment & VisionThis is a "North Star" prototype built using an AI-native design workflow to establish the ideal user experience for future engineering implementation.Live Prototype: https://lesleyps.github.io/donkey-donkey/Document Version: 1.0.5 - Senior Product Design Portfolio.
